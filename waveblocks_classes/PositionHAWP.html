class WaveBlocksND.PositionHAWP[source]¶

This class implements the computation of the action of the position operator applied to a Hagedorn wavepacket .

apply_position(wavepacket, *, component=None, as_packet=True)[source]¶

Compute the effect of the position operator on the basis functions of a component of the Hagedorn wavepacket .

Parameters:
  • wavepacket (A HagedornWavepacketBase subclass instance.) – The wavepacket containing .
  • component (Integer or None.) – The index of the component .
  • as_packet (Boolean, default is True.) – Whether to return a full packet.
Returns:

A list of length or a single pair of extended basis shapes and new coefficients . If requested, copies of the original wavepacket are returned with these new values set. There are packets, one for each space variable component of the position operator.
